02 · Featured Work

WhatsTheRate — AI Currency Intelligence

An autonomous, end-to-end ML pipeline that forecasts EUR/GBP exchange rates using a Random Forest ensemble, providing actionable, data-driven transfer recommendations.

Python 3.10 Streamlit Scikit-learn yfinance Pandas GitHub Actions Joblib Matplotlib
System Architecture (Three layers):
1. Automated Data & ML Pipeline: A daily GitHub Actions workflow fetches fresh EUR/GBP data, engineers technical indicators (e.g., 30-day moving average), and retrains a RandomForestRegressor to ensure the model stays adaptive.
2. Intelligence Logic: Forecasts the next session's rate using engineered features with model persistence via joblib. A threshold-based recommendation engine outputs STRONG BUY, WAIT, or NEUTRAL signals.
3. Streamlit Web App: A multi-page interface featuring a live Dashboard, AI Predictions, and an Optimal Transfer tool that statistically identifies the cheapest weekday for currency exchange.
  • Forecast Accuracy: Achieved a Mean Absolute Error of 0.00299 using lag signals and volatility features.
  • DevOps Automation: A fully automated CI/CD pipeline handles daily retraining and model commits, drastically reducing manual toil.
  • HCI Integration: Prioritizes immediate system feedback, recognition-based navigation, and clear system state indicators for a low cognitive load.
